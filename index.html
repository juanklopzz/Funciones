Etiquetas extras

Etiqueta pre

La etiqueta pre sirve para respetar espacios y lineas en bblanco al contrario de la etiqueta parrafo

Hola me llamo Juan Carlos tengo 16 años 

 Hola     soy
        Juan Carlos      tengo   16
        años

Etiqueta abr

La etiqueta abr se utiliza para las abreviaciones

Segun datos de la OMS el consumo excesivo de azucar provoca daños severos en la salud de los niños.

Etiqueta bdo

Si deseas cambiar la dirección de un texto, se utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

Al millón porque al cien cualquera por: Juan Carlos López

Etiqueta small

La etiquita small hace el texto más pequeño

Este texto esta dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subindice se debe utilizar la etiqueta sub

Etiqueta time

Etiqueta address

Para esécificar al navegador que el texto se refiere a una dirección se utiliza la etiqueta address
